# 微信开发者工具配置指南

## 1. 安装微信开发者工具

1. 前往 [微信开发者工具下载页面](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 下载对应操作系统的稳定版（Stable）
3. 安装后使用微信扫码登录

## 2. 导入项目

### 构建小程序

```bash
# 安装依赖
npm install

# 构建小程序产物
npm run build:weapp
```

### 在开发者工具中导入

1. 打开微信开发者工具，点击**导入项目**
2. 目录选择项目根目录下的 `dist/` 文件夹
3. AppID 填写你的小程序 AppID（见下方"测试号 vs 注册号"说明）
4. 点击**导入**

### 开发模式（热更新）

```bash
npm run dev:weapp
```

运行后开发者工具会自动检测 `dist/` 目录变更并刷新预览。

## 3. 云开发环境配置

### 开通云开发

1. 在开发者工具中，点击左上角**云开发**按钮
2. 首次使用需要开通，选择按量付费模式（有免费额度）
3. 创建一个环境，记录环境 ID（如 `guilidao-xxxx`）

### 初始化数据库集合

在云开发控制台的**数据库**页面，手动创建以下集合：

| 集合名 | 说明 |
|--------|------|
| `users` | 用户信息 |
| `products` | 产品目录 |
| `tours` | 团次信息 |
| `orders` | 订单数据 |
| `order_notes` | 订单备注 |
| `delivery_locations` | 配送地点 |

### 创建云函数

项目中的云函数位于 `src/cloud/` 目录。

1. 在开发者工具中右键 `cloudfunctions` 目录
2. 选择**同步云函数列表**
3. 右键每个云函数文件夹，选择**创建并部署: 云端安装依赖**

需要部署的云函数：
- `login` — 获取用户 OpenID

### 配置环境 ID

确保 `src/app.tsx` 中 `initCloud()` 使用的环境 ID 与你创建的一致。如需指定环境 ID，修改 `src/services/db.ts`：

```typescript
cloud.init({ env: '你的环境ID', traceUser: true });
```

## 4. 测试号 vs 注册号

### 测试号（推荐开发阶段使用）

- 在开发者工具中选择**测试号**即可，无需注册
- 限制：不能使用部分高级 API，不能上传发布
- 云开发功能可正常使用

### 注册号

- 前往 [微信公众平台](https://mp.weixin.qq.com/) 注册小程序账号
- 获取 AppID 后在项目设置中填入
- 需要在后台设置中添加开发者

## 5. H5 开发模式

如果不需要微信开发者工具，可以使用 H5 模式进行 UI 开发：

```bash
npm run dev:h5
```

H5 模式使用内置 mock 数据，无需云开发环境，可在浏览器中直接预览和调试页面。

## 6. 常见问题排查

### "找不到 app.json"

确保导入的是 `dist/` 目录而非项目根目录。先运行 `npm run build:weapp` 生成构建产物。

### 云开发初始化失败

- 检查是否已开通云开发
- 确认环境 ID 配置正确
- 确认 `project.config.json` 中有正确的 `cloudfunctionRoot` 配置

### "login" 云函数调用失败

- 确认 `login` 云函数已部署
- 在云开发控制台检查云函数日志

### 编译报错 "未找到 xxx 组件"

```bash
# 清理并重新构建
rm -rf dist/
npm run build:weapp
```

### 真机调试白屏

- 检查是否有未兼容的 ES6+ 语法
- 在开发者工具**详情 > 本地设置**中开启"ES6 转 ES5"
- 确认网络请求域名已在后台配置白名单

### 开发者工具卡顿 / 预览慢

- 关闭不需要的插件
- 在**设置 > 通用设置**中关闭"自动保存"
- 清理模拟器缓存：**工具 > 清缓存 > 全部清除**
